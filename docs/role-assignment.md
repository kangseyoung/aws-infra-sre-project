# Team Role Assignment

| Member | Main Role | Detailed Scope | Deliverables |
| --- | --- | --- | --- |
| 강세영 | Observability / SRE + Network | CloudWatch metrics and alarms, log visibility, network traffic flow explanation, architecture support | Monitoring notes, alarm checklist, traffic flow summary |
| 권태욱 | Compute / Platform | EC2 provisioning, Docker installation, Nginx runtime, ALB backend connection, instance operation notes | EC2 setup record, container run guide, service verification notes |
| 김태우 | Terraform / IaC | Terraform folder ownership, provider and variable design, draft resource codification, IaC documentation | Terraform draft files, variable/output notes, IaC follow-up plan |
| 박찬혁 | Security + Observability Support | Security Group review, IAM role review, CloudWatch support, final checklist validation | Security checklist, IAM notes, observability support notes |

## Collaboration Rules
- Record all manual setup steps in the relevant `infra/manual-setup/` files.
- Capture screenshots for major AWS resources and save them in `diagrams/` if needed.
- Update meeting notes after each team sync.
- Use runbooks when testing failure scenarios.
