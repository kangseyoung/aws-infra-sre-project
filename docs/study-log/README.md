# Study Log Guide

이 폴더는 팀원들이 매일 공부한 내용을 기록하는 공간입니다.

공부기록은 완벽한 정리본이 아니어도 됩니다. 오늘 공부한 내용, GPT에게 물어본 질문, 내가 이해한 내용, 헷갈리는 부분, 내일 할 일을 남기는 것이 목적입니다.

## 작성 방법

1. 각자 본인 폴더에 날짜별 Markdown 파일을 만듭니다.
2. 파일명은 날짜만 사용합니다.
3. 템플릿을 복사해서 작성합니다.

파일명 예시:

```text
docs/study-log/kangseyoung/2026-06-29.md
docs/study-log/kwon-taeuk/2026-06-29.md
docs/study-log/kim-taewoo/2026-06-29.md
docs/study-log/park-chanhyuk/2026-06-29.md
```

템플릿:

```text
templates/study-log-template.md
```

## 작성할 내용

- 담당 파트
- 오늘 공부한 개념
- 내가 이해한 내용
- GPT에게 물어본 질문
- 새로 알게 된 점
- Must / Optional 후보
- 아직 헷갈리는 부분
- 내일 할 일
- 참고한 자료

## Notes

- 민감한 정보는 절대 올리지 않습니다.
- AWS Access Key, Secret Key, `.pem`, `.env`, `terraform.tfstate`, `.terraform/`은 절대 커밋하지 않습니다.
- 비용이 발생할 수 있는 리소스는 실습 후 반드시 삭제합니다.
- 공부기록은 복붙보다 본인이 이해한 말로 정리하는 것을 우선합니다.
