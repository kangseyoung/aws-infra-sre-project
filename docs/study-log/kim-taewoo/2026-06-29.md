## 1. 담당 파트

AWS 인프라 프로젝트에서 **Terraform / IaC** 파트를 맡았다.

Compute나 Network처럼 독립적으로 기능을 구현하는 역할이라기보다는  
 AWS 리소스를 코드(IaC)로 관리하는 역할이다.

아마 금요일부터 여유가 조금 생길 것 같다.

AWS SAA 강의와 Docker&Kubernetes 강의를 통해 인프라 이해도를 높일 계획이다.

## 6. Must / Optional 후보

### Must 후보

VPC, Subnet, Route Table, Internet Gateway, Security Group 정도의 핵심 리소스를 코드화

### Optional 후보

EC2, ALB, CloudWatch, IAM 등을 Terraform으로 관리하거나 모듈화
